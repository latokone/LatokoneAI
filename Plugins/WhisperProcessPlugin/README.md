### WhisperProcessPlugin
This folder is part of the LatokoneAI project and contains the implementation of the WhisperProcessPlugin, which integrates Whisper‑based speech‑to‑text processing into the LatokoneAI orchestration framework.

### 🧩 Purpose
The WhisperProcessPlugin extends LatokoneAI with support for:

* Running Whisper speech‑to‑text models locally
* Handling audio input and transcription
* Integrating STT into multi‑model pipelines (e.g., STT → LLM → TTS)

For more information, see [Whisper.net](https://github.com/sandrohanea/whisper.net)
