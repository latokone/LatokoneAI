### ImageGeneration Plugin
This folder contains the ImageGeneration plugin for the LatokoneAI framework. It provides components related to image‑generation workflows.

### 🧩 Purpose
The ImageGeneration plugin is part of the modular plugin architecture of LatokoneAI.
It allows the engine to:

* Load and manage image‑generation models
* Provide a unified interface for image‑generation tasks
* Integrate with the orchestration engine for scheduling and resource management

For more information see [StableDiffusion.NET](https://github.com/DarthAffe/StableDiffusion.NET)
