### YoloPluginProcess
This folder is part of the LatokoneAI project and contains the implementation of the YoloPluginProcess, a plugin that integrates YOLO‑based object detection into the LatokoneAI orchestration framework.

### 🧩 PurposeThe YoloPluginProcess extends LatokoneAI with support for:
The YoloPluginProcess extends LatokoneAI with support for:

* Running YOLO object‑detection models locally
* Processing images and returning bounding boxes, labels, and confidence scores
* Integrating object detection into multi‑model pipelines (e.g., vision → LLM reasoning)

For more information, see [YoloDotNet](https://github.com/NickSwardh/YoloDotNet)
