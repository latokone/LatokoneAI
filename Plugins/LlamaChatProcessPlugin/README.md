### LlamaChatProcessPlugin
This folder is part of the LatokoneAI project and contains the implementation of the LlamaChatProcessPlugin, a plugin that integrates Llama‑based chat processing into the LatokoneAI orchestration framework.

### 🧩 Purpose
The LlamaChatProcessPlugin extends LatokoneAI with support for running Llama‑based chat models as part of local AI pipelines.
It fits into the modular plugin architecture, allowing the engine to:

* Load and manage Llama chat models
* Handle message passing and inference
* Integrate with the orchestration engine for scheduling and resource control

For more information, see [LLamaSharp](https://github.com/SciSharp/LLamaSharp)
