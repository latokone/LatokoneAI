### KokoroProcessPlugin
This directory is part of the LatokoneAI project and contains the files for the KokoroProcessPlugin, a plugin used to integrate Kokoro‑based TTS processing into the LatokoneAI orchestration framework.

### 🧩 Purpose
The KokoroProcessPlugin extends LatokoneAI with support for Kokoro‑based text‑to‑speech processing.
It fits into the modular plugin architecture, allowing the engine to load and run Kokoro processes as part of local AI pipelines.

For more information see [KokoroSharp](https://github.com/Lyrcaxis/KokoroSharp)
